Computational Thinking

Computational Thinking is divided up into four main components:

  • Abstraction
  • Decomposition
  • Pattern Recognition
  • Algorithms

Abstraction

Simplifying a problem by removing unnecessary components

Decomposition

Breaking down a complex problem into smaller, more manageable sub-problems

Pattern Recognition

Finding similarities or patterns within a data set or sequence to simplify understanding and problem-solving. The three tasks you want to complete when using pattern recognition are to, get data, make rules and check to see if A implies B

Algorithms

Developing step-by-step solutions or rules to solve problems. Similarly, to Data recognition, using an algorithm requires the use of the three separate tasks, sequence, iteration and selection
